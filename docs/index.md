# Shoveler Docs

Shoveler is a lightweight [DuckDB](https://duckdb.org) workbench built for learning and teaching SQL.


## Start Here

1. [Quickstart (10 minutes)](quickstart.md)
2. [Core Concepts](concepts.md)
3. [Troubleshooting](troubleshooting.md)

## Who These Docs Are For

- Students learning SQL and basic database workflows
- Teachers running in-class demos and lab sessions
- Self-learners who want a simple desktop DuckDB tool

## What You Can Do in Shoveler

- Open a DuckDB file or create a blank in-memory database
- Run SQL using `F5` or `Ctrl+Enter`
- Run selected SQL only, or run all SQL in the editor
- Explore tables and columns in the schema panel
- Double-click a table name to insert it into your SQL editor
- Export query results to CSV from the results panel

## Suggested Learning Path

1. Complete the quickstart once from top to bottom.
2. Review the concepts page to understand why features behave the way they do.
3. Use troubleshooting when setup or runtime behavior is unclear.
